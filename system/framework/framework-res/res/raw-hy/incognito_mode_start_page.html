Դուք անցաք անանուն կերպի. Այս պատուհանում դիտվող էջերը չեն հայտնվի Ձեր զննիչի կամ որոնման պատմությունում և անանուն պատուհանը փակելուց հետո չեն թողնի այլ հետքեր, օրինակ` թխուկներ: Այնուամենայնիվ, ցանկացած ներբեռնված նիշքերը կամ ստեղծված էջանիշները կպահպանվեն:

Անանուն կերպը չի ազդում այլ մարդկանց, սպասարկիչների կամ ծրագրաշարի վարքին: Զգուշացեք հետևյալից.

  • Ձեր մասին տեղեկություն հավաքող կամ համօգտագործող կայքեր
  • Ձեր այցելված էջերը հետագծող Համացանցի ծառայություն մատուցողներից կամ աշխատողներից
  • Վնասարար ծրագրաշարերից, որոնք հետագծում են Ձեր ստեղնահարվածները` անվճար սմայլիկների փոխարեն
  • Գաղտնի գործակալների վերահսկումից
  • Ձեր հետևում կանգնող մարդկանցից
