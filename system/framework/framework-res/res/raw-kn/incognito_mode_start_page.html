ನೀವು ಅಜ್ಞಾತರಾಗಿದ್ದೀರಿ. ಈ ವಿಂಡೋನಲ್ಲಿ ನೀವು ನೋಡುವ ಪುಟಗಳು ನಿಮ್ಮ ಬ್ರೌಸರ್ ಹಿಸ್ಟರಿ ಅಥವಾ ಸರ್ಚ್ ಹಿಸ್ಟರಿಯಲ್ಲಿ ಕಾಣಿಸುವುದಿಲ್ಲ ಮತ್ತು ಅವು ಬೇರೆ ಸುಳಿವುಗಳನ್ನು ಅಂದರೆ ಕುಕೀಗಳಂಥವುಗಳನ್ನು, ನೀವು ಅಜ್ಞಾತ ಟ್ಯಾಬ್ ಮುಚ್ಚಿದ ಬಳಿಕ ಉಳಿಸುವುದಿಲ್ಲ. ನೀವು ಡೌನ್‌ಲೋಡ್ ಮಾಡುವ ಯಾವುದೇ ಫೈಲ್‌ಗಳು ಅಥವಾ ನೀವು ರಚಿಸುವ ಬುಕ್‌ಮಾರ್ಕ್‌ಗಳನ್ನು ಉಳಿಸಲಾಗುತ್ತದೆ, ಆದರೆ.

ಅಜ್ಞಾತರಾಗುವುದು ಬೇರೆ ಜನ, ಸರ್ವರ್‌ಗಳು ಅಥವಾ ಸಾಫ್ಟ್‌ವೇರ್‌ನ ನಡವಳಿಕೆಯ ಮೇಲೆ ಪ್ರಭಾವ ಬೀರುವುದಿಲ್ಲ. ಇವುಗಳ ಬಗ್ಗೆ ಎಚ್ಚರವಿರಲಿ:

  • ನಿಮ್ಮ ಬಗ್ಗೆ ಮಾಹಿತಿಯನ್ನು ಸಂಗ್ರಹಿಸುವ ಅಥವಾ ಹಂಚಿಕೊಳ್ಳುವ ವೆಬ್‌ಸೈಟ್‌ಗಳು
  • ನೀವು ಭೇಟಿ ನೀಡುವ ಪುಟಗಳ ಜಾಡು ಹಿಡಿಯುವ ಇಂಟರ್‌ನೆಟ್ ಸೇವಾ ಕಂಪನಿಗಳು ಅಥವಾ ಉದ್ಯೋಗಿಗಳು
  • ಉಚಿತ ಸ್ಮೈಲಿಗಳ ನೆವದಲ್ಲಿ ನಿಮ್ಮ ಕೀ ಸ್ಟ್ರೋಕ್‌ಗಳ ಜಾಡು ಇರಿಸುವ ಕೊಳಕು ಸಾಫ್ಟ್‌ವೇರ್
  • ಸೀಕ್ರೆಟ್ ಏಜೆಂಟರ ಕಣ್ಗಾವಲು
  • ನಿಮ್ಮ ಹಿಂದೆ ಇರುವ ಜನರು
