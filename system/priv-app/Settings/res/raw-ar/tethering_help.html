التوصيل عبر USB

يمكنك توصيل هاتفك بالكمبيوتر باستخدام كبل USB، وذلك لمشاركة اتصال الإنترنت الخاص بهاتفك مع الكمبيوتر
  • يعمل التوصيل مع أنظمة التشغيل Windows Vista، Windows 7، Windows XP، وLinux

  • لا يمكنك تثبيت بطاقة SD الخاصة بهاتفك على الكمبيوتر عند التوصيل عبر USB

  • لمزيد من المعلومات، بما في ذلك استخدام التوصيل عبر USB مع أنظمة التشغيل الأخرى، راجع http://www.android.com/tether#usb

التوصيل عبر Bluetooth

يمكنك توصيل جهاز Android الخاص بك بجهاز الكمبيوتر عبر اتصال Bluetooth، وذلك من أجل مشاركة اتصال الإنترنت المتوفر بجهازك مع جهاز الكمبيوتر لديك.
  • يصلُح التوصيل عبر Bluetooth لأي جهاز كمبيوتر أو غيره من الأجهزة التي يمكنها الحصول على اتصال بشبكة الإنترنت عن طريق تقنية Bluetooth

  • لمزيد من المعلومات، تفضل بزيارة http://www.android.com/tether#bluetooth

اتصال Wi-Fi المحمول

يمكنك تحويل هاتفك إلى نقطة اتصال Wi-Fi محمول، وذلك لمشاركة اتصال الإنترنت الخاص بهاتفك مع جهاز كمبيوتر واحد أو أكثر، أو أجهزة أخرى
  • إذا كان هاتفك يعمل كنقطة اتصال Wi-Fi، لا يمكنك استخدام تطبيقات الهاتف للاتصال بالإنترنت عبر اتصال Wi-Fi

  • يمكنك تهيئة نقطة الاتصال مع إعدادات نقطة اتصال Wi-Fi

  • لمزيد من المعلومات، تفضل بزيارة http://www.android.com/tether#wifi
