التوصيل عبر USB

يمكنك توصيل هاتفك بالكمبيوتر باستخدام كبل USB، وذلك لمشاركة اتصال الإنترنت الخاص بهاتفك مع الكمبيوتر
  • يعمل التوصيل مع أنظمة التشغيل Windows Vista، Windows 7، Windows XP، وLinux

  • لا يمكنك تثبيت بطاقة SD الخاصة بهاتفك على الكمبيوتر عند التوصيل عبر USB

  • لمزيد من المعلومات، بما في ذلك استخدام التوصيل عبر USB مع أنظمة التشغيل الأخرى، راجع http://www.android.com/tether#usb
