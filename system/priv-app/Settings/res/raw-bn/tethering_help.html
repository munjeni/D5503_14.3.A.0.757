USB টেথরিং

আপনার ফোনের ইন্টারনেট সংযোগ কম্পিউটারের সাথে অংশীদারি করতে একটি USB কেবলের মাধ্যমে আপনি আপনার ফোনটিকে কম্পিউটারের সাথে আটকাতে পারেন
  • টেথরিং Windows XP, Windows Vista, Windows 7, এবং Linux-এর সাথে কাজ করে

  • USB আটকানো অবস্থায় আপনি আপনার ফোনের SD কার্ডকে আপনার কম্পিউটারে মাউন্ট করতে পারবেন না

  • অন্যান্য OS-এর সাথে USB টেথরিং ব্যবহার সহ আরও তথ্যের জন্য, http://www.android.com/tether#usb দেখুন

Bluetooth সংযোগকরণ

আপনার ডিভাইসের ইন্টারনেট সংযোগ আপনার কম্পিউটারের সাথে অংশীদারি করতে আপনি Bluetooth সংযোগের মাধ্যেমে আপনার Android যন্ত্রটিকে কম্পিউটারের সাথে সংযোগ করতে পারেন
  • Bluetooth সংযোগকরণ যেকোনো কম্পিউটার বা অন্যান্য যন্ত্রের সাথে কাজ করতে পারে যা Bluetooth-এর মাধ্যমে একটি ইন্টারনেট সংযোগ অর্জন করতে পারে

  • আরও তথ্যের জন্য, http://www.android.com/tether#bluetooth দেখুন

পোর্টেবেল Wi-Fi হটস্পট

আপনার ফোনের ইন্টারনেট সংযোগ একটি বা তার বেশি কম্পিউটারের সাথে বা যন্ত্রের সাথে অংশীদারি করতে আপনি নিজের ফোনটিকে একটি পোর্টেবেল Wi-Fi হটস্পটে পরিবর্তিত করতে পারেন
  • আপনার ফোন যখন একটি Wi-Fi হটস্পট হিসাবে কাজ করে তখন, সেটির Wi-Fi সংয়োগের মাধ্যেমে ইন্টারনেট অ্যাক্সেস করার জন্য আপনি ফোনের অ্যাপ্লিকেশন ব্যবহার করতে পারবেন না

  • Wi-Fi হটস্পট সেটিংসের মাধ্যমে আপনি হটস্পট কনফিগার করুন

  • আরও তথ্যের জন্য, দেখুন http://www.android.com/tether#wifi
