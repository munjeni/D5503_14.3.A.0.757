Ancoratge USB

Podeu ancorar el telèfon a l'ordinador amb un cable USB per compartir la connexió a Internet del telèfon amb l'ordinador
  • L'ancoratge funciona amb Windows XP, Windows Vista, Windows 7 iLinux

  • Podeu instal·lar la targeta SD del telèfon a l'ordenador quan s'han ancorat amb un cable USB

  • Per obtenir més informació, inclosa la d'utilitzar l'ancoratge USB amb altres OS, vegeu http://www.android.com/tether#usb

Lligam a Internet amb Bluetooth

Podeu lligar el vostre dispositiu Android a l'ordinador amb una connexió Bluetooth, per compartir la connexió a Internet del dispositiu amb l'ordinador.
  • El lligam a Internet amb Bluetooth funciona amb qualsevol ordinador o dispositiu que pugui obtenir una connexió a Internet mitjançant Bluetooth

  • Per obtenir més informació, aneu http://www.android.com/tether#bluetooth

Punt de connexió Wi-Fi portàtil

Podeu convertir el telèfon en un punt de connexió portàtil Wi-Fi, per compartir la connexió a Internet del telèfon amb un o més ordinadors o altres dispositius
  • Quan el telèfon funciona com a punt de connexió Wi-Fi, no podeu utilitzar les aplicacions de l'ordinador per accedir la Internet amb la seva connexió Wi-Fi

  • Configureu el punt de connexió amb els paràmetres del punt de connexió Wi-Fi

  • Per obtenir més infromació, visiteu http://www.android.com/tether#wifi
