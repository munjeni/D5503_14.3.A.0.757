Σύνδεση μέσω κινητής συσκευής με USB

Μπορείτε να συνδέσετε το τηλέφωνο με τον υπολογιστή σας με ένα καλώδιο USB, για κοινή χρήση της σύνδεσης στο Διαδίκτυο του τηλεφώνου σας με τον υπολογιστή σας
  • Η σύνδεση μέσω κινητής συσκευής λειτουργεί με Windows XP, Windows Vista, Windows 7 και Linux

  • Δεν είναι δυνατό να τοποθετήσετε την κάρτα SD του τηλεφώνου στον υπολογιστή σας, όταν υπάρχει σύνδεση μέσω κινητής συσκευής με USB

  • Για περισσότερες πληροφορίες, συμπεριλαμβανομένης εκείνων για τη σύνδεση μέσω κινητής συσκευής με USB με άλλα λειτουργικά συστήματα, ανατρέξτε στη διεύθυνση http://www.android.com/tether#usb

Tethering μέσω Bluetooth

Μπορείτε να κάνετε χρήση του tethering μέσω της συσκευής Android για τον υπολογιστή σας μέσω σύνδεσης Bluetooth, για να κάνετε κοινή χρήση με τον υπολογιστή σας της σύνδεσης της συσκευής σας στο Internet.
  • Το tethering μέσω Bluetooth λειτουργεί με κάθε υπολογιστή ή άλλη συσκευή που μπορεί να συνδεθεί στο Internet μέσω Bluetooth.

  • Για περισσότερες πληροφορίες, δείτε http://www.android.com/tether#bluetooth

Σύνδεση μέσω κινητής συσκευής με Wi-Fi

Μπορείτε να μετατρέψετε το τηλέφωνό σας σε ένα σημείο πρόσβασης Wi-Fi, για κοινή χρήση της σύνδεσης στο Διαδίκτυο του τηλεφώνου σας με έναν ή περισσότερους υπολογιστές ή άλλες συσκευές
  • Όταν το τηλέφωνό σας λειτουργεί ως σημείο πρόσβασης, δεν είναι δυνατό να χρησιμοποιήσετε τις εφαρμογές του τηλεφώνου σας για να αποκτήσετε πρόσβαση στο Διαδίκτυο μέσω της σύνδεσης Wi-Fi

  • Μπορείτε να ρυθμίσετε ένα σημείο πρόσβασης Wi-Fi στις ρυθμίσεις σύνδεσης μέσω κινητής συσκευής με Wi-Fi

  • Για περισσότερες πληροφορίες, επισκεφθείτε τη διεύθυνση http://www.android.com/tether#wifi
