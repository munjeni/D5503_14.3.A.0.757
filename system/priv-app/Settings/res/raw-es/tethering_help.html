Anclaje de USB

Puedes anclar el teléfono a tu ordenador con un cable USB para compartir la conexión a Internet del teléfono con el equipo.
  • El anclaje funciona con Windows XP, Windows Vista, Windows 7 y Linux.

  • Puedes montar la tarjeta SD del teléfono en el equipo una vez realizado el anclaje de USB.

  • Para obtener más información, incluido el uso del anclaje de USB con otros sistemas operativos, consulta la página http://www.android.com/tether#usb.

Uso compartido de Internet por Bluetooth

Puede compartir Internet entre su dispositivo Android y su ordenador a través de una conexión de Bluetooth, y así podrá compartir la conexión a Internet del dispositivo con el ordenador.
  • El uso compartido de Internet por Bluetooth funciona con cualquier ordenador u otro dispositivo que pueda obtener una conexión a Internet a través de Bluetooth

  • Para obtener más información, consulte http://www.android.com/tether#bluetooth

Anclaje a red Wi-Fi

Puedes convertir el teléfono en un punto de acceso Wi-Fi para compartir la conexión a Internet del teléfono con uno o con varios equipos o dispositivos.
  • Cuando el teléfono actúe como un punto de acceso, no podrás utilizar sus aplicaciones para acceder a Internet a través de la conexión Wi-Fi del dispositivo.

  • Para configurar un punto de acceso Wi-Fi, accede a la configuración de anclaje a red Wi-Fi.

  • Para obtener más información, consulta la página http://www.android.com/tether#wifi.
