USB bidezko amarratzea

USB kable batekin amarra dezakezu telefonoa ordenagailura, telefonoaren Interneteko konexioa ordenagailuarekin partekatzeko
  • Windows XP, Windows Vista, Windows 7 eta Linux sistema eragileekin funtzionatzen du amarratzeak

  • Ezin duzu telefonoaren SD txartelik sartu ordenagailuan USBa amarratuta dagoenean

  • Informazio gehiago eskuratzeko, beste sistema eragile batzurekin USB bidezko amarratzea barne, ikusi http://www.android.com/tether#usb helbidea

Bluetooth bidezko konexioa

Android gailua Bluetooth bidez konekta dezakezu ordegailura, gailuren Interneteko konexioa ordenagailuarekin partekatzeko
  • Bluetooth bidezko konexioak edozein ordenagailurekin edo Bluetooth bidez Interneteko konexioa eskura dezakeen beste edozein gailurekin funtzionatzen du

  • Informazio gehiago eskuratzeko, ikus http://www.android.com/tether#bluetooth helbidea

Wi-Fi publiko eramangarria

Telefonoa Wi-Fi publiko eramangarri bihur dezakezu, telefonoaren Interneteko konexioa ordenagailu batekin edo gehiagorekin edo beste gailu batzuekin partekatzeko
  • Telefonoa Wi-Fi publiko gisa dabilenean, ezin dituzu Wi-Fi konexioaren bidez Internetera sartzeko telefonoaren aplikazioak erabili

  • Wi-Fi publikoaren ezarpenekin konfiguratzen duuzu Wi-Fi publikoa

  • Informazio gehiago eskuratzeko, bisitatu http://www.android.com/tether#wifi helbidea
