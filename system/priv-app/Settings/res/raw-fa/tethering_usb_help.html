اتصال USB

شما می توانید تلفن خود را با استفاده از یک کابل USB به رایانه متصل کنید و اتصال اینترنت تلفن خود را با رایانه به اشتراک بگذارید
  • این نوع اتصال با Windows XP،‏ Windows Vista‏،‏ Windows 7 و Linux کار می کند

  • هنگامی که تلفن را با کابل USB به رایانه متصل نموده اید نمی توانید از کارت SD تلفن خود در رایانه استفاده کنید

  • جهت کسب اطلاعات بیشتر از جمله درباره نحوه استفاده از اتصال USB به سایر سیستم های عامل، به نشانی http://www.android.com/tether#usb مراجعه کنید
