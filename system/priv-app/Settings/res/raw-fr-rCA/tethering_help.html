Tethering USB

Vous pouvez raccorder votre téléphone à votre ordinateur avec un câble USB, pour partager la connection Internet de votre téléphone avec votre ordinateur
  • Tethering fonctionne sous Windows XP, Windows Vista, Windows 7, et Linux

  • Vous ne pouvez pas insérer la carte SD de votre téléphone sur votre ordinateur lorsqu'il est branché via le câble USB

  • Pour plus d'informations, y compris sur l'utilisation de Tethering USB, visitez http://www.android.com/tether#usb

Fonction modem du Bluetooth

Vous pouvez utiliser comme modem votre appareil Android à votre ordinateur au moyen d'une connexion Bluetooth, afin de partager la connexion Internet de votre appareil avec votre ordinateur.
  • La fonction modem du Bluetooth est compatible avec tout ordinateur ou appareil qui peut obtenir une connexion Internet par le biais d'un Bluetooth

  • Pour obtenir davantage d'informations, ouvrez le lien http://www.android.com/tether#bluetooth

Point d'accès Wi-Fi mobile

Vous pouvez transformer votre téléphone en point d'accès Wi-Fi mobile pour partager la connexion Internet de votre téléphone avec un ou plusieurs ordinateurs, ou bien encore d'autres appareils
  • Lorsque vous vous servez de votre téléphone en tant que point d'accès Wi-Fi, vous ne pouvez pas utiliser ses applications d'accès à Internet via sa connexion Wi-Fi

  • Vous configurez le point d'accès par les paramètres de point d'accès Wi-Fi

  • Pour plus d'informations, visitez http://www.android.com/tether#wifi
