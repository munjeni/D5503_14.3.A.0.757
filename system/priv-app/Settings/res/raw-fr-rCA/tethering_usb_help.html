Tethering USB

Vous pouvez raccorder votre téléphone à votre ordinateur avec un câble USB, pour partager la connection Internet de votre téléphone avec l'ordinateur
  • Tethering fonctionne sous Windows XP, Windows Vista, Windows 7, et Linux

  • Vous ne pouvez pas insérer la carte SD de votre téléphone dans votre ordinateur lorsqu'il est branché via le câble USB

  • Pour plus d'informations, y compris sur l'utilisation de Tethering USB, visitez http://www.android.com/tether#usb
