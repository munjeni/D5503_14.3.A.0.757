USB टेदरिंग

आप अपने फ़ोन के इंटरनेट कनेक्शन को अपने कंप्यूटर से साझा करने के लिए अपने फ़ोन को USB केबल की सहायता से अपने कंप्यूटर से टेदर कर सकते हैं
  • टेदरिंग Windows XP, Windows Vista, Windows 7, और Linux के साथ कार्य करता है

  • जब USB को टेदर किया गया हो, तो आप अपने फ़ोन के SD कार्ड को अपने कंप्यूटर पर माउंट नहीं कर सकते

  • अन्य OS के साथ USB टेदरिंग के उपयोग सहित अधिक सूचना के लिए, http://www.android.com/tether#usb देखें
