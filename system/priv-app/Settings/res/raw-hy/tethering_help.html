USB կապում

Կարող եք կապել Ձեր Android սարքը Ձեր համակարգչին USB մալուխի միջոցով` Ձեր սարքի Համացանցի կապակցումը Ձեր համակարգչի հետ համօգտագործելու համար
  • USB կապումն աշխատում է Windows XP, Windows Vista, Windows 7 և Linux-ի հետ

  • Եթե Ձեր սարքն ունի SD քարտ կամ USB պահեստ, չեք կարող ապասարել այն Ձեր համակարգչում, երբ USB-ն կպված է

  • Լրացուցիչ տեեկության համար, ներառյալ այլ ԳՀ-ների հետ USB կապման օգտագործման վերաբերյալ, տես http://www.android.com/tether#usb

Bluetooth-ի կապում

Կարող եք կապել Ձեր Android սարքը Ձեր համակարգչին bluetooth կապակցման միջոցով` Ձեր սարքի Համացանցի կապակցումը Ձեր համակարգչի հետ համօգտագործելու համար
  • Bluetooth կապումն աշխատում է ցանկացած այլ համակարգչի կամ սարքի հետ, որը կարող է ունենալ կապակցում Համացանցին` Bluetooth-ի միջոցով

  • Լրացուցիչ տեղեկության համար, տես http://www.android.com/tether#bluetooth

Հեռացվելի Wi-Fi hotspot

Կարող եք միացնել Ձեր Android սարքը հեռացվելի Wi-Fi hotspot-ին` Ձեր Android սարքի կապակցումը Համացանցին մեկ կամ ավելի համակարգիչների կամ այլ սարքերի հետ համօգտագործելու համար
  • Երբ Ձեր սարքը ծառայում է որպես Wi-Fi hotspot, չեք կարող օգտագործել Ձեր հեռախոսի գործադիրները` իր Wi-Fi Կապակցման միջոցով Համացանց մատչելու համար (սակայն, ի հարկե, մնում եք կապակցված Ձեր բջջային տվյալների ցանցի միջոցով)

  • Դուք կազմաձևեցիք hotspot-ը Wi-Fi hotspot-ի կարգավորումների համաձայն

  • Լրացուցիչ տեղեկության համար, տես http://www.android.com/tether#wifi
