USB tethering

Anda dapat meranggit telepon ke komputer menggunakan kabel USB, untuk berbagi sambungan Internet telepon dengan komputer
  • Tethering dapat dilakukan dengan Windows XP, Windows Vista, Windows 7, dan Linux

  • Anda tidak dapat memasang kartu SD telepon pada komputer ketika USB sedang tethering

  • Untuk mendapatkan informasi selengkapnya termasuk cara menggunakan USB tethering pada SO lain, lihat http://www.android.com/tether#usb

Peranggitan Bluetooth

Anda dapat meranggitkan/menambatkan perangkat Android Anda ke komputer lewat sambungan Bluetooth, untuk membagikan sambungan internet perangkat Anda dengan komputer
  • Peranggitan/penambatan Bluetooth berfungsi di komputer mana pun atau perangkat lain yang dapat memperoleh sambungan Internet lewat Bluetooth

  • Untuk informasi lebih lanjut, lihat http://www.android.com/tether#bluetooth

Hotspot portabel Wi-Fi

Anda dapat mengaktifkan telepon menjadi hotspot Wi-Fi portabel untuk membagikan sambungan internet telepon dengan salah satu atau beberapa komputer atau perangkat yang lain
  • Saat telepon Anda berfungsi sebagai hotspot Wi-Fi, Anda dapat menggunakan aplikasi telepon untuk mengakses Internet melalui sambungan Wi-Fi-nya

  • Anda mengonfigurasikan hotspot dengan setelan hotspot Wi-Fi

  • Untuk informasi selengkapnya, kunjungi http://www.android.com/tether#wifi
