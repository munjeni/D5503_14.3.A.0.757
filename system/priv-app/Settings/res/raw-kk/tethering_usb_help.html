USB арқылы байланыстыру

Телефонның интернет байланысын компьютермен ортақтастыру үшін телефонды компьютерге USB кабелінің көмегімен байланыстыруға болады
  • Байланыстыру Windows XP, Windows Vista, Windows 7 және Linux жүйелерінде жұмыс істейді

  • USB арқылы байланыстырған кезде телефонның SD картасын компьютерден ашу мүмкін болмайды

  • Басқа амалдық жүйелерде USB арқылы байланыстыруды қоса, қосымша ақпарат алу үшін http://www.android.com/tether#usb мекенжайын қараңыз
