USB ಯಿಂದ ಸಂಪರ್ಕಿಸುವುದು

ನೀವು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರಿನೊಂದಿಗೆ ನಿಮ್ಮ ಸಾಧನದ ಇಂಟರ್‌ನೆಟ್ ಸಂಪರ್ಕವನ್ನು ಹಂಚಿಕೊಳ್ಳುವುದಕ್ಕಾಗಿ USB ಕೇಬಲ್ ಬಳಕೆ ಮಾಡುವ ಮೂಲಕ ನಿಮ್ಮ Android ಸಾಧನವನ್ನು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರಿಗೆ ಸಂಪರ್ಕಿಸಬಹುದು
  • USB ಟೀದರಿಂಗ್ Windows XP, Windows Vista, Windows 7, ಮತ್ತು Linux ಜೊತೆ ಕೆಲಸ ಮಾಡುವುದು

  • ನಿಮ್ಮ ಸಾಧನವು SD ಕಾರ್ಡ್‌ ಅಥವಾ USB ಸಂಗ್ರಹಣೆಯನ್ನು ಹೊಂದಿದ್ದಲ್ಲಿ, USB ಸಂಪರ್ಕಗೊಂಡಿರುವಾಗ ನೀವು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರಿನಲ್ಲಿ ಮೌಂಟ್ ಮಾಡಲು ಸಾಧ್ಯವಾಗುವುದಿಲ್ಲ

  • ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ, ಬೇರೆ OS ಗಳ ಜೊತೆ USB ಸಂಪರ್ಕವನ್ನು ಬಳಕೆ ಮಾಡಲು, http://www.android.com/tether#usb ನೋಡಿ

Bluetooth ಮೂಲಕ ಸಂಪರ್ಕಿಸುವುದು

ನೀವು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರಿನೊಂದಿಗೆ ನಿಮ್ಮ ಸಾಧನದ ಇಂಟರ್‌ನೆಟ್ ಸಂಪರ್ಕವನ್ನು ಹಂಚಿಕೊಳ್ಳುವುದಕ್ಕಾಗಿ Bluetooth ಸಂಪರ್ಕದ ಮೂಲಕ ನಿಮ್ಮ Android ಸಾಧನವನ್ನು ನಿಮ್ಮ ಕಂಪ್ಯೂಟರಿಗೆ ಸಂಪರ್ಕಿಸಬಹುದು
  • Bluetooth ಸಂಪರ್ಕವು Bluetooth ಮೂಲಕ ಇಂಟರ್‌ನೆಟ್ ಸಂಪರ್ಕ ಪಡೆಯಬಹುದಾದ ಯಾವುದೇ ಕಂಪ್ಯೂಟರ್ ಅಥವಾ ಬೇರೆ ಸಾಧನದೊಂದಿಗೆ ಕೆಲಸ ಮಾಡುತ್ತದೆ

  • ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ http://www.android.com/tether#bluetooth ನೋಡಿ

ಪೋರ್ಟಬಲ್ Wi-Fi ಹಾಟ್‌ಸ್ಪಾಟ್

ನೀವು ನಿಮ್ಮ Android ಸಾಧನವನ್ನು ಪೋರ್ಟಬಲ್ Wi-Fi ಹಾಟ್‌ಸ್ಪಾಟ್ ಆಗಿ ಬದಲಿಸಬಹುದು, ನಿಮ್ಮ Android ಸಾಧನದ ಇಂಟರ್‌ನೆಟ್ ಸಂಪರ್ಕವನ್ನು ಒಂದು ಅಥವಾ ಹೆಚ್ಚಿನ ಕಂಪ್ಯೂಟರ್‌ಗಳು ಅಥವಾ ಬೇರೆ ಸಾಧನಗಳೊಂದಿಗೆ ಹಂಚಿಕೊಳ್ಳಬಹುದು
  • ನಿಮ್ಮ ಸಾಧನವು Wi-Fi ಹಾಟ್‌ಸ್ಪಾಟ್ ಆಗಿ ಕೆಲಸ ಮಾಡುತ್ತಿರುವಾಗ, ಅದರ Wi-Fi ಸಂಪರ್ಕದ ಮೂಲಕ ಇಂಟರ್‌ನೆಟ್ ಪ್ರವೇಶಿಸುವುದಕ್ಕಾಗಿ ನೀವು ನಿಮ್ಮ ಫೋನಿನ ಅಪ್ಲಿಕೇಷನ್‌ಗಳನ್ನು ಬಳಕೆ ಮಾಡಲು ಸಾಧ್ಯವಿಲ್ಲ (ಆದರೆ ಖಂಡಿತವಾಗಿ ನಿಮ್ಮ ಮೊಬೈಲ್ ಡೇಟಾ ನೆಟ್‌ವರ್ಕ್ ಮೂಲಕ ಇಂಟರ್‌ನೆಟ್‌ ಸಂಪರ್ಕದಲ್ಲಿ ಉಳಿಯಬಹುದು)

  • Wi-Fi ಹಾಟ್‌ಸ್ಪಾಟ್ ಸೆಟ್ಟಿಂಗ್ಸ್ ಜೊತೆ ಹಾಟ್‌ಸ್ಪಾಟ್ ಅನ್ನು ಕಾನ್ಫಿಗರ್ ಮಾಡಿ

  • ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ http://www.android.com/tether#wifi ಗೆ ಭೇಟಿಕೊಡಿ
