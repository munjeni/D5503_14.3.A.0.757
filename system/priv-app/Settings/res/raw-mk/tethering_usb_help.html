Поврзување со USB

Вие може да го поврзете вашиот телефон со вашиот компјутер со USB кабел, за да ја споделите интернет конекцијата на вашиот телефон со вашиот компјутер
  • Поврзувањето работи со Windows XP, Windows Vista, Windows 7 и Linux

  • Вие не може да ја поставите SD картичката од вашиот телефон на вашиот компјутер кога е приклучен USB

  • За повеќе информации, вклучувајќи користење на USB поврзување со други ОС, видете http://www.android.com/tether#usb
