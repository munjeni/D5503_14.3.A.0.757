USB ടെതർ ചെയ്യൽ

നിങ്ങളുടെ ഉപകരണത്തിന്‍റെ ഇന്‍റർനെറ്റ് കണക്ഷൻ നിങ്ങളുടെ കമ്പ്യൂട്ടറുമായി പങ്കിടുന്നതിന് ഒരു USB കേബിൾ ഉപയോഗിച്ച് നിങ്ങളുടെ കമ്പ്യൂട്ടറിലേക്ക് നിങ്ങളുടെ Android ഉപകരണം ടെതർ ചെയ്യാനാകും
  • USB ടെതർ ചെയ്യൽ Windows XP, Windows Vista, Windows 7, Linux എന്നിവയിൽ പ്രവർത്തിക്കും

  • നിങ്ങളുടെ ഉപകരണത്തിന് ഒരു SD കാർഡ് അല്ലെങ്കിൽ USB സ്റ്റോറേജ് ഉണ്ടെങ്കിൽ, USB ടെതർ ചെയ്യുമ്പോൾ നിങ്ങളുടെ കമ്പ്യൂട്ടറിൽ ഇത് മൗണ്ട് ചെയ്യാനാകില്ല

  • മറ്റ് OSകൾ ഉപയോഗിച്ച് USB ടെതർ ചെയ്യൽ ഉപയോഗിക്കുന്ന ഉൾപ്പെടയുള്ള കൂടുതൽ വിവരങ്ങൾക്ക്, കാണുക http://www.android.com/tether#usb

Bluetooth ടെതർ ചെയ്യൽ

നിങ്ങളുടെ ഉപകരണത്തിന്‍റെ ഇന്‍റർനെറ്റ് കണക്ഷൻ നിങ്ങളുടെ കമ്പ്യൂട്ടറുമായി പങ്കിടുന്നതിന് ഒരു Bluetooth കണക്ഷൻ വഴി നിങ്ങളുടെ കമ്പ്യൂട്ടറിലേക്ക് നിങ്ങളുടെ Android ഉപകരണം ടെതർ ചെയ്യാനാകും
  • Bluetooth വഴി ഇന്‍റർനെറ്റ് കണക്ഷൻ ലഭിക്കുന്ന ഏതൊരു കമ്പ്യൂട്ടറിലും അല്ലെങ്കിൽ മറ്റ് ഉപകരണത്തിലും Bluetooth ടെതർ ചെയ്യൽ പ്രവർത്തിക്കും

  • കൂടുതൽ വിവരങ്ങൾക്ക്, കാണുക http://www.android.com/tether#bluetooth

പോർട്ടബിൾ Wi-Fi ഹോട്ട്‌സ്പോട്ട്

നിങ്ങളുടെ Android ഉപകരണത്തിന്‍റെ ഇന്‍റർനെറ്റ് കണക്ഷൻ ഒന്നോ അതിൽ കൂടുതലോ കമ്പ്യൂട്ടറുകളുമായോ മറ്റ് ഉപകരണങ്ങളുമായോ പങ്കിടുന്നതിന് നിങ്ങൾക്ക് നിങ്ങളുടെ Android ഉപകരണം പോർട്ടബിൾ Wi-Fi ഹോട്ട്‌സ്പോട്ടിലേക്ക് മാറ്റാനാകും
  • നിങ്ങളുടെ ഉപകരണം ഒരു Wi-Fi ഹോട്ട്‌സ്പോട്ടായി പ്രവർത്തിക്കുമ്പോൾ, Wi-Fi കണക്ഷൻ വഴി ഇന്‍റർനെറ്റിൽ പ്രവേശിക്കുന്നതിന് നിങ്ങൾക്ക് നിങ്ങളുടെ ഫോണിന്‍റെ ആപ്ലിക്കേഷനുകൾ ഉപയോഗിക്കാനാകും (എന്നാൽ, നിങ്ങൾ തീർച്ചയായും നിങ്ങളുടെ മൊബൈൽ ഡാറ്റ നെറ്റ്‌വർക്ക് വഴി ഇന്‍റർനെറ്റിലേക്ക് കണക്ടുചെയ്തിരിക്കുന്നത് തുടരും)

  • Wi-Fi ഹോട്ട്‌സ്പോട്ട് ക്രമീകരണങ്ങൾ ഉപയോഗിച്ച് നിങ്ങൾ ഹോട്ട്‌സ്പോട്ട് കോൺഫിഗർ ചെയ്തു

  • കൂടുതൽ വിവരങ്ങൾക്ക്, സന്ദർശിക്കുക http://www.android.com/tether#wifi
