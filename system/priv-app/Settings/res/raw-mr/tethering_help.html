USB टेदरिंग

आपल्या उपकरणाचे इंटरनेट कनेक्शन आपल्या संगणकाबरोबर शेअर करण्यासाठी आपण आपले Android उपकरण USB केबलच्या साहाय्याने आपल्या संगणकाशी टेदर करू शकता
  • USB टेदरिंग आपल्या Windows XP, Windows Vista, Windows 7, आणि Linux बरोबर काम करते

  • आपल्या उपकरणात SD कार्ड किंवा USB संचय असेल, तर USB टेदर केलेला असताना आपण तो आपल्या संगणकावर माऊंट करू शकत नाही

  • इतर OSs शी USB टेदरिंग वापरण्यासह, अधिक माहितीसाठी, http://www.android.com/tether#usb पहा

Bluetooth टेदरिंग

आपल्या उपकरणाचे इंटरनेट कनेक्शन आपल्या संगणकाबरोबर शेअर करण्यासाठी, आपण आपले Android उपकरण Bluetooth कनेक्शनमार्फत आपल्या संगणकाशी टेदर करू शकता
  • Bluetooth टेदरिंग कोणत्याही संगणकाबरोबर किंवा इतर उपकरणाबरोबर काम करते ज्याला Bluetooth मार्फत इंटरनेट कनेक्शन आहे

  • अधिक माहितीसाठी http://www.android.com/tether#bluetooth पहा

पोर्टेबल Wi-Fi हॉटस्पॉट

आपल्या Android उपकरणाचे इंटरनेट कनेक्शन एक किंवा जास्त संगणक किंवा इतर उपकरणांबरोबर शेअर करण्यासाठी, आपण आपले Android उपकरण पोर्टेबल Wi-Fi हॉटस्पॉटमध्ये रूपांतरित करू शकता
  • आपले उपकरण Wi-Fi हॉटस्पॉट स्वरूपात काम करीत असेल, तेव्हा त्याच्या Wi-Fi कनेक्शनमार्फत इंटरनेटमध्ये प्रवेश करण्यासाठी आपण आपल्या फोनचे अनुप्रयोग वापरू शकत नाही (परंतु अर्थातच आपण आपल्या मोबाईल डेटा नेटवर्कमार्फत नेटवर्कशी कनेक्ट केलेले रहाता)

  • Wi-Fi हॉटस्पॉट सेटिंग्जनी आपण हॉटस्पॉट संरूपित करता

  • अधिक माहितीसाठी http://www.android.com/tether#wifi पहा
