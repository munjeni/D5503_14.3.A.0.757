Tether USB

Anda boleh menghubungkan telefon anda ke komputer anda melalui kabel USB, untuk berkongsi sambungan internet telefon anda dengan komputer anda
  • Penghubungan boleh berfungsi dengan Windows XP, Windows Vista, Windows 7, dan Linux

  • Anda tidak boleh memasang kad SD telefon anda pada komputer anda semasa ia dihubungkan melalui USB

  • Untuk mendapatkan maklumat lanjut, termasuk menggunakan penghubungan USB dengan OS lain, lihat http://www.android.com/tether#usb

Tether Bluetooth

Anda boleh menghubungkan peranti Android anda ke komputer anda melalui sambungan Bluetooth, untuk berkongsi sambungan Internet peranti anda dengan komputer anda
  • Tether Bluetooth berfungsi dengan sebarang komputer atau peranti lain yang boleh mendapatkan sambungan Internet melalui Bluetooth

  • Untuk mendapatkan maklumat lanjut, lawati http://www.android.com/tether#bluetooth

Hotspot Wi-Fi mudah alih

Anda boleh menukar telefon anda menjadi hotspot Wi-Fi mudah alih, untuk berkongsi sambungan internet telefon anda dengan satu atau lebih komputer atau peranti lain
  • Semasa telefon anda berfungsi sebagai hotspot Wi-Fi, anda tidak boleh menggunakan aplikasi telefon anda untuk mengakses internet melalui sambungan Wi-Finya

  • Anda boleh mengkonfigurasi hotspot dengan tetapan hotspot Wi-Fi

  • Untuk mendapatkan maklumat lanjut, lawati http://www.android.com/tether#wifi
