USB-подключение

Вы можете подключить телефон к компьютеру с помощью USB-кабеля и предоставить компьютеру интернет-подключение, установленное посредством телефона.
  • Такое подключение возможно для ОС Windows XP, Windows Vista, Windows 7 и Linux.

  • При USB-подключении вы не сможете получить доступ к SD-карте телефона с компьютера.

  • Дополнительная информация и сведения об USB-подключении в других ОС приведена на странице http://www.android.com/tether#usb.

Связь по Bluetooth

Можно установить связь между устройством Android и компьютером по Bluetooth, чтобы использовать имеющееся на устройстве подключение к Интернету также на компьютере.
  • Связь по Bluetooth работает с любым компьютером или другим устройством, которое может подключаться к Интернету по Bluetooth

  • Для получения дополнительной информации посетите веб-сайт http://www.android.com/tether#bluetooth

Подключение Wi-Fi

Телефон можно использовать как точку доступа Wi-Fi. В этом случае несколько компьютеров или других устройств смогут использовать интернет-подключение, установленное посредством телефона.
  • Когда телефон выступает в роли точки доступа, его приложения не смогут получить доступ к Интернету через подключение Wi-Fi.

  • Включить режим общего доступа к Интернету через Wi-Fi можно в настройках подключения Wi-Fi.

  • Дополнительная информация приведена на странице http://www.android.com/tether#wifi.
