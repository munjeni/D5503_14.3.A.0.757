USB டீதரிங்

உங்கள் சாதனத்தின் இணைய இணைப்பை உங்கள் கணினியுடன் பகிர, ஒரு USB கேபிள் மூலம் உங்கள் கணினியை உங்கள் Android சாதனத்தை டீதர் செய்யலாம்
  • USB டீதரிங் Windows XP, Windows Vista, Windows 7, மற்றும் Linux உடன் செயலாற்றும்

  • உங்கள் சாதனத்தில் ஒரு SD கார்டு அல்லது USB சேமிப்பகம் இருந்தால், USB டீதர் செய்யப்பட்ட போது அதனை உங்கள் கணினியில் ஏற்ற முடியாது

  • மற்ற OSகளுடன் USB டீதரிங் பயன்படுத்துவது உள்ளிட்ட கூடுதல் தகவலுக்கு, http://www.android.com/tether#usb பர்க்கவும்

Bluetooth டீதரிங்

உங்கள் சாதனத்தின் இணைய இணைப்பை உங்கள் கணினியுடன் பகிர, ஒரு Bluetooth இணைப்பு மூலம் உங்கள் கணினியை உங்கள் Android சாதனத்தை டீதர் செய்யலாம்
  • Bluetooth வழியாக ஒரு இணைய இணைப்பைப் பெறக்கூடிய ஒரு கணினி அல்லது மற்ற சாதனத்துடன் Bluetooth டீதரிங் செயல்படும்

  • மேலும் தகவலுக்கு, http://www.android.com/tether#bluetooth பார்க்கவும்

சிறிய Wi-Fi hotspot

உங்கள் Android சாதனத்தின் இணைய இணைப்பை ஒன்று அல்லது கூடுதல் கணினிகள் அல்லது மற்ற சாதனங்களுடன் பகிர, உங்கள் Android சாதனத்தை ஒரு சிறிய Wi-Fi hotspotபாட்டாக மாற்றலாம்
  • ஒரு Wi-Fi hotspotடாக உங்கள் சாதனம் சேவையாற்றுகையில், அதன் Wi-Fi இணைப்பு வழியாக இணையத்தை அணுக உங்கள் ஃபோனின் பயன்பாடுகளை நீங்கள் பயன்படுத்த முடியாது (ஆனால் உங்கள் மொபைல் தரவு நெட்வொர்க் மூலம் இணையத்துடன் தொடர்ந்து இணைந்திருக்கலாம்)

  • Wi-Fi hotspot அமைப்புகளுடன் ஹாட்ஸ்பாட்டை உள்ளமைக்கவும்

  • மேலும் தகவலுக்கு, http://www.android.com/tether#wifi செல்லவும்
