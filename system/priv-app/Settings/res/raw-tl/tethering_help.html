USB na pag-tether

Maaari mong i-tether ang iyong telepono sa iyong computer gamit ang isang USB na kable, upang ibahagi ang internet connection ng telepono sa iyong computer
  • Gumagana ang pag-tether sa Windows XP, Windows Vista, Windows 7, at Linux

  • Hindi mo mailalagay ang SD card ng iyong telepono sa iyong computer kapag naka-tether ang USB

  • Para sa higit na impormasyon, kabilang ang paggamit ng USB na pag-tether sa ibang mga OS, tingnan ang http://www.android.com/tether#usb

Bluetooth tethering

Maaari mong i-tether ang iyong Android device sa iyong computer sa pamamagitan ng koneksyon sa Bluetooth, upang maibahagi ang koneksyon sa Internet ng iyong device sa iyong computer
  • Gumagana ang Bluetooth tethering sa anumang computer o iba pang device na makakakuha ng koneksyon sa Internet sa pamamagitan ng Bluetooth

  • Para sa higit pang impormasyon, tingnan ang http://www.android.com/tether#bluetooth

Portable na Wi-Fi hotspot

Maaari mong gawing portable na Wi-Fi hotspot ang iyong telepono, upang maibahagi ang internet connection ng iyong telepono sa isa o higit pang mga computer o mga aparato
  • Kapag nagsisilbi bilang Wi-Fi hotspot ang iyong telepono, hindi mo magagamit ang mga application ng iyong telepono upang ma-access ang internet sa pamamagitan ng Wi-Fi na connection nito

  • Iko-configure mo ang hotspot gamit ang Wi-Fi hotspot na mga setting

  • Para sa higit na impormasyon, bisitahin ang http://www.android.com/tether#wifi
