USB 網絡共享

您可使用 USB 傳輸線，將手機與電腦共享網絡，即讓您的電腦分享手機的互聯網連接
  • 網絡共享可在 Windows XP、Windows Vista、Windows 7 及 Linux 上使用

  • USB 網絡共享時，您無法將手機的 SD 卡掛接電腦

  • 欲知詳情，包括在其他操作系統使用 USB 網絡共享的資料，請瀏覽 http://www.android.com/tether#usb

藍牙網絡共享

您可經由藍牙連接，將 Android 裝置與電腦共享網絡，即讓您的電腦分享裝置的互聯網連接
  • 藍牙網絡共享可通過能夠經由藍牙獲得互聯網連接的任何電腦或其他裝置來工作

  • 欲知詳情，請參考 http://www.android.com/tether#bluetooth

可攜式Wi-Fi無線基地台

您可將手機設成可攜式Wi-Fi無線基地台，然後與一部或多電腦或其他裝置分享手機的互聯網連接。
  • 當手機用作Wi-Fi無線基地台時，您無法使用手機的應用程式，經由Wi-Fi連接訪問互聯網（但是您可以繼續經由您的手機流動網絡保持互聯網連接）

  • 您可以在可攜式Wi-Fi無線基地台設定中設定無線基地台

  • 欲知詳情，請瀏覽 http://www.android.com/tether#wifi
