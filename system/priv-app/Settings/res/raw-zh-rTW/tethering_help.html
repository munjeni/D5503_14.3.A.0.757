USB 數據連線

您可以使用 USB 纜線讓手機和電腦連線，使電腦可以共用手機的網際網路連線
  • 支援數據連線的作業系統包括 Windows XP、Windows Vista、Windows 7 及 Linux

  • 使用 USB 數據連線時，您無法將手機的 SD 卡插到電腦上

  • 如需更多資訊，包括在其他作業系統中使用 USB 數據連線的資訊，請參閱 http://www.android.com/tether#usb

藍牙數據連線

您可透過藍牙連線將 Android 裝置數據連線至電腦，與您的電腦共享裝置的網際網路連線
  • 藍牙數據連線適用於任何電腦和其他可透過藍牙取得網際網路連線的裝置

  • 如需更多資訊，請參閱 http://www.android.com/tether#bluetooth

可攜式Wi-Fi無線基地台

您可以將手機設成可攜式Wi-Fi無線基地台，讓一台或多台電腦或其他裝置可以共用手機的網際網路連線
  • 將手機當作無線基地台使用時，手機的應用程式將無法透過Wi-Fi連線訪問網際網路（但您可以繼續透過您的手機行動網路連線訪問網際網路）

  • 您可以在Wi-Fi無線基地台設定中配置無線基地台

  • 如需更多資訊，請造訪 http://www.android.com/tether#wifi
